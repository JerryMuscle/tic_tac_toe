# 3目並べ(tic_tac_toe)
基本的な3目並べのwebアプリです。

## インストールと起動方法
```
git clone https://github.com/JerryMuscle/tic_tac_toe.git
cd tic_tac_toe
docker compose up

http://localhost:8080　に接続
```
